# vuetify-pwa

## Includes
- PWA
- Vue2.5+
- Vue-cli3
- Vue-Router
- Vuex
- Vuetify
- Typescript
- Pug
- Stylus

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
